Setting up your coding environment


Setting up your environment is very Simple, all you need is a Google account to About Colab : Colab is a virtual runtime where basically Google lends you a Computer on the web to log in into Google Colab. You don't need download any module on your local machine for that. Run your code on Colab, with GPU support, all for FREE. Colab runs in almost any browser you use. Throughout all the tutorials, we would be using Colab, so it is necessary to set it up before we proceed further.

1. Go to colab.research.google.com
2. Login with your Google account
3. Create a new Notebook
(File > New notebook)

By default, it would be a .pynb (python notebook) File and infact we will be coding all of our Machine Learning models in the Python Language. General knowledge about python would be all that you need.


Using Colab GPU

GPU processes tensors(data) a lot faster than cpu, so for large models, it could even take days on cpu power whereas the GPU does it a lot faster. At times, using the GPU would be quite helpful, but not every Machine Learning task you're going to do now would require the GPU, so use the GPU ONLY when necessary. Tutorials where GPU use is suggested are marked accordingly.

To use the GPU, Go to Runtime > Change runtime type > Hardware accelerator > GPU > Save



For now that's all you need to know to get started.

Now that your environment is set up, lets start from the very basics of Machine Learning.
